# FTB Quests 任务系统

## 配置文件位置
- `config/ftbquests/quests/chapters/` — 每个 .snbt 文件一个章节
- `config/ftbquests/quests/chapter_groups/` — 章节组（标签页）

## 指令
- `/ftbquests reload` — 重载任务文件
- `/ftbquests open` — 打开任务书
- `/ftbquests complete <id>` — 完成任务

## ID 规则
- 所有 ID 是 16 进制数字字符串
- ID 最大长度 16 位 hex
- 首字符必须在 0-7 范围

## 章节结构（当前整合包）
4 组：
- 根目录：序章-植物魔法, 仓储
- 发展：植物魔法→自然灵气→机械动力→新生魔艺→血魔法→魔法艺术→铁魔法→黑暗仪式→神秘学→神化→无尽贪婪→神秘辣条→更多→冰火传说
- 探索：维度巡礼→Boss图鉴→装备与武器→饰品→魂石
- 终局：终焉→永恒之门→深渊之座→终焉之证
- 法术修炼：铁魔法学校体系

## FTB Quests 奖励类型
- `item` — 物品奖励
- `command` — 执行指令
- `xp` — 经验奖励
- `quest` — 解锁其他任务（依赖）
- `reward_table` — 随机奖励表

## 常用的 NBT 键
- title: 任务标题
- description: 任务描述
- icon: 图标物品
- x/y: 位置坐标
- quests: 子任务列表
- tasks: 完成条件
- rewards: 奖励列表
