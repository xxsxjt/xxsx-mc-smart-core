# Blood Magic 血魔法

## 核心机制
- **LP（生命印记）**：血魔法的能量单位
- **祭坛（Blood Altar）**：核心方块，需升级
- **灵魂网络（Soul Network）**：跨维度 LP 共享

## 祭坛等级（6级）
1. 基础祭坛 — 石砖
2. 增强祭坛 — 红石+荧石
3. 高级祭坛 — 血之符文
4. 究极祭坛 — 强化符文
5. 极限祭坛 — 本源符文
6. 巅峰祭坛 — 源质符文

## 主要指令
- `/bloodmagic bind` — 绑定玩家到祭坛
- `/bloodmagic orb` — 查看当前印记信息
- `/bloodmagic cleanse` — 清除绑定

## 炼金术桌
用于合成物品，需要 LP 和原料。
配方 JSON 在 `kubejs/data/bloodmagic/recipes/` 中。

## 仪式
通过仪式石（Ritual Stone）激活。
- 需要激活石（Activation Crystal）
- 消耗 LP
- 不同仪式有不同的符文结构
