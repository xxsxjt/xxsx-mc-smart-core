## 正文

xxsx的智能核心是一个将 AI 大语言模型接入 Minecraft 的 Forge 模组，通过 `/ai` 命令实现自然语言操控游戏。

玩家使用自然语言描述需求，AI 自主分析意图并执行对应的 Minecraft 指令，支持多轮迭代完成复杂任务。
内置免费共享 API Key，开箱即用，同时支持个人自定义 API。

主要功能包括：

- 自然语言 AI 对话，AI 自动执行 Minecraft 指令并反馈结果；
- 游戏状态查询，AI 可查询玩家信息、方块类型、物品属性、合成配方、附近实体等；
- PMX 体素建筑，解析 MMD 模型文件并通过表面体素化与 CIELAB 颜色匹配转换为 Minecraft 方块建筑；
- 知识库系统，jar 内置模组知识库且支持外置扩展，AI 自动按需查询；
- 个人 API 配置，每位玩家可独立设置自己的 API Key 与模型；
- 多模型切换，支持 Agnes、DeepSeek 及自定义 API 供应商的游戏内切换；
- 上下文自动压缩，超限时可选择截断或 AI 总结；
- 会话持久化，同一玩家重连后对话记录自动恢复。


## 指令列表

- `/ai <消息>` — 与 AI 自然语言对话；
- `/ai build <PMX路径> [比例]` — 解析 PMX 模型生成体素建筑；
- `/ai api <供应商>` — 切换 AI 供应商；
- `/ai model <模型名>` — 切换 AI 模型；
- `/ai addmodel <地址> <Key> <模型> [上下文长度]` — 添加个人 API；
- `/ai stop` — 中断当前任务；
- `/ai clear` — 清除对话记录。
